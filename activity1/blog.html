My Web Dev Journey

Week 1: Getting Started with HTML

Published on: June 09, 2026

This week marks the start of my web development journey in college. I've had some prior experience with HTML and CSS before, but I always felt like there were gaps in my understanding. Now I finally have the chance to fill those gaps and learn it the right way.

What excites me most is realizing how much more there is to HTML than I thought. I knew the basics, but concepts like semantic structure and accessibility were things I never really paid attention to before. I'm looking forward to seeing how much further I can push my skills from here.

The best way to learn is to build.
